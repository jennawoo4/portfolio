Some of the similarities between Python and Scratch is that they use similar functions. For example, Scratch and Python both have a function to increase a certain variable value by a certain number. They can both carry out similar tasks, although they are in different forms. Scratch and Python are different in that Scratch uses preprogrammed blocks that you can click together to execute a code, while in Python you have to type the code yourself. You also use Scratch on the Scratch website, but you can use Python on the Command Prompt on your laptop.
